### 메이플 인벤토리
- 리액트를 연습하는데 Todolist 만들기가 싫어서 인벤토리를 만들어보았다.
- 이미지 구해서 넣는게 노가다가 짙었다
- 현재 메이플 인벤토리와 내가 새로 디자인한 인벤토리를 비교해서 가지고 놀 수 있게 만들었다.
  - 현재 메이플 인벤은 조금 더 바뀌었지만...

### 실행 방법
```
yarn        // yarn 설치
yarn dev    // dev 실행
```
